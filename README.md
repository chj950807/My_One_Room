# 📔주요 기능 소개
* 일반 로그인, 로그아웃, 회원가입, 회원탈퇴 (DB : Firebase)
* 구글 소셜 로그인 가능합니다. 카카오 소셜 로그인은 OAuth 인가코드 받는 부분까지 구현 
* 별점체크(Star Rating) 를 통한 점수 평가
* 게시글 삭제, 답글 기능
* Daum 주소 api를 이용해 검색 기능 제공
* Tailwind CSS 를 이용한 반응형 디자인 (mobile, desktop)
